# 📚 References

## Overview

This document contains the primary references, datasets, government reports, academic publications, industry resources, and analytical materials used in the research project:

> **Data-Driven Contractor Selection and Housing Affordability Analysis in Madurai**

The references support the study's investigation into contractor performance, housing affordability, construction industry practices, and data-driven decision-making through Exploratory Data Analysis (EDA).

These sources provided valuable information for dataset development, contractor verification, housing research, and analytical interpretation.
